# RELARAWEBI - Chat Application
Chat Application using React 19 and Laravel 12 for practice


## 🚀 Features Implemented

### Find Friends
- Send Friend Request
- Find Friends

### ✅ Core Messaging
- One-to-one conversations
- Real-time message delivery
- Optimized database structure

### ✅ Unread Message Count (Real-Time)
- Per-conversation unread counter
- Automatically increments for recipients
- Resets instantly when conversation is opened

### ✅ Read Receipts (✔✔)
- Messages marked as read when chat is opened
- Real-time broadcast to other participants
- WhatsApp-style UX behavior

### ✅ Clean Architecture
- Action classes for business logic
- DTOs for data transport
- Events for real-time communication
- Thin controllers (senior-style Laravel)

## 🧱 Tech Stack

| Layer | Technology |
|-----|-----------|
| Backend | Laravel |
| Real-Time | Laravel Reverb |
